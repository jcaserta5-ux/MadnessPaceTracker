# Madness Pace Tracker

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/jcaserta5-ux/MadnessPaceTracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MadnessPaceTracker
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Features
- **Real-time Pace Tracking:** Track your pace in real-time as you play.
- **User-Friendly Interface:** Intuitive and easy to navigate UI.
- **Statistics Dashboard:** View detailed statistics of your performance.
- **Customizable Settings:** Configure the application settings to fit your needs.
- **Multi-Platform Support:** Compatible with various platforms including mobile and desktop.
